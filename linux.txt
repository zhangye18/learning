linux 安装git

1.安装git
	yum install git
2.查看git状态
	git --version
3.配置git用户信息
	git config --global user.name "名字"
	git config --global user.email "邮箱"
4.查看SSH服务状态
	ps -e|grep sshd
	sshd 表示ssh-server已启动
5.生成SSH KEY
    ls -al ~/.ssh
	查看ssh key 是否存在，存在忽略
	ssh-keygen -t rsa -C "邮箱"
注意 ssh-keygen 中间不能有空格，否则会报错"Bad escape character ‘ygen’"

6.查看SSH KEY
cd /root/.ssh	


安装 lrzsz
yum install lrzsz
